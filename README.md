# Meme Generator (Docker + Flask)

This project is a simple Meme Generator built with **Python**, **Flask**, and **Pillow**.  
The goal is to create a web application that runs fully inside a **Docker container**.

## Features
- Upload any image through a web form
- Enter top and bottom text (classic meme format)
- Text automatically fits the width of the image
- Generates a meme and displays it in the browser
- Saves created memes in `static/generated/`
